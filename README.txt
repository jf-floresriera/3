# Portal Inmobiliario

Este proyecto es una aplicación web desarrollada con Flask para mostrar proyectos inmobiliarios.

## Estructura

- `/static/` → CSS, imágenes, videos
- `/templates/` → HTMLs de las vistas
- `app.py` → Servidor Flask
- `requirements.txt` → Dependencias

## Instalación local

```bash
pip install -r requirements.txt
python app.py
```

Luego abre en tu navegador: http://127.0.0.1:5000/
